# 🎌 Japanese Vocabulary Quiz

A terminal-based quiz game built in Python to help study Japanese vocabulary.

## About
I built this project to combine my interest in Japanese language and culture
with my Python programming skills. It was my first independent project after
earning my PCEP (Certified Entry-Level Python Programmer) certification.

## Features
- 10 Japanese vocabulary words with English translations
- Score tracking and percentage calculation
- Performance rating based on your score
- Replay system to keep practicing

## Concepts Used
- Variables, functions, and loops
- Dictionaries for word storage
- Input validation
- Modular code structure

## How to Run
1. Make sure Python 3 is installed
2. Clone this repository
3. Run `python vocab_quiz.py` in your terminal

This is the first project in a series I'm building to strengthen my Python
